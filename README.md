
# SQL Business Insights Project

## Project Overview

This project analyzes retail sales data using MySQL to generate business insights and support data-driven decision-making.

The objective was to explore customer behavior, product performance, and country-wise sales trends using SQL queries.

---

## Tools Used

* MySQL
* MySQL Workbench
* SQL

---

## Dataset

Online Retail Dataset

The dataset contains:

* Customer Information
* Product Details
* Invoice Transactions
* Quantity Sold
* Unit Price
* Country Information

---

## Business Questions

### 1. Who are the top customers by revenue?

Objective:
Identify high-value customers contributing the most revenue.

### 2. Which products generate the highest revenue?

Objective:
Determine the best-performing products.

### 3. Which products sell the highest quantity?

Objective:
Analyze product demand.

### 4. Which countries generate the most revenue?

Objective:
Identify top-performing markets.

### 5. What is the average order value?

Objective:
Understand customer spending behavior.

### 6. How do products rank by revenue?

Objective:
Use Window Functions to rank products based on revenue contribution.

## Key Findings

### Customer Insights

* A small number of customers generated a significant portion of total revenue.
* Repeat customers contributed substantially to sales performance.

### Product Insights

* Top-selling products accounted for a large share of overall revenue.
* Certain products generated high revenue despite lower sales volume.

### Country Insights

* The United Kingdom generated the highest revenue.
* Several international markets contributed significantly to sales growth.
